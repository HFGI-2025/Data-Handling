    Assignment DH4

    Assignment Background:

    This assignment contains four sections of questions that build upon and incorporate methods, techniques and concepts taught and demonstrated in the lesson materials. Each section corresponds to the Data Handling lessons, 1-4.

    All questions aim to consider biological context. Timely submission will result in your assignment being returned within 14 days of the submission deadline. Marked Jupyter Notebook manuscripts will be returned with comments from your marker. If you receive a pass or distinction, your marker will attach the model solutions for this assignment via the open pull request on your lesson's GitHub repository.

    This assignment aims to test the application of knowledge, skills and techniques taught across the four Data Handling lessons in this module.

    DH1

    Question 1

    A.

    Read the cervical cancer dataset into the workspace. You can find the csv file provided in the Data folder.

    • Import Pandas and use the read_csv function

    • display the first 10 rows using .head()
    Hint: You can reference a file without the full path using dots to indicate relative path navigation. Using one . indicates the file/folder is in the same directory you are currently in, while using .. indicates the path starts in the parent directory of the one you're in.

    Example: To access the data here use the path "../Data/cervical_cancer.csv"
    B.

    Explore the dataframe to understand the data:
    • Find out how many rows and columns are there? Print the result in a formatted string

    • How many columns contain floating point numbers? Print your answer
    Hint: Use equivalence == to 'float64' to obtain a list of True (1) and False (0) values

    Question 2

    A.

    Analyse the number of smokers in the dataset:

    • Find out how many of the subjects (rows) are smokers?

    • Calculate the percentage of smokers.

    • Round the answer to 1 decimal place.
    Give both answers as a formatted string.
    B.

    Investigate the spread of age in the dataset:
    • Plot the age distribution as a histogram using matplotlib, using 50 bins
    C.

    Investigate the average number of sexual partners for those who have had at least one diagnosis of an STD:
    • Filter the dataset to only include data where diagnosis of STDs is 1 or more
      • Use the column "STDs: Number of diagnosis" to do this

    • From the filtered dataset calculate the mean and standard deviation of number of sexual partners.
